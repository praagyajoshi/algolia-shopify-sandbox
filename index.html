Algolia x Shopify Codesandbox

Welcome to the Codesandbox for the Shopify integration!
Check out the search page, or a sample collection page.

Replicating a config

To replicate a config from a live store:

  • Edit this sandbox
  • Replace the contents of algolia_config.js with that from the store
  • In index.html, update the "main-content" class to a value that's available on the actual store (either the search page or collections page as applicable)

Previewing a collection page

To preview a collection page:

  • Edit this sandbox
  • Update template_algolia_current_collection_id to mention the actual collection ID
  • Navigate to /collections/collection-handle

Some notes

  • This sandbox is only meant to replicate the Algolia widgets on the search and collection page of a Shopify store and not the whole store itself. Any customisations done to the widgets on a Shopify store will not be available here automatically.
